# Cross Player 1.2.3

This patch release fixes player spacing and control placement across desktop and mobile layouts.

Highlights:
- Removed extra view padding that was leaving wasted space around the player, especially in desktop layouts.
- Adjusted video sizing and alignment so the player uses the available space more cleanly instead of appearing to float with a fake bottom gutter.
- Repositioned the mobile progress controls so they stay above Obsidian's bottom navigation without sitting unnecessarily high on screen.
