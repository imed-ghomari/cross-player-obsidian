# Cross Player 1.2.0

This feature release improves the mobile player experience and fullscreen layout.

Highlights:
- Removed the hidden inset around the player so the media view fills the pane more cleanly, especially in light mode.
- Improved mobile fullscreen sizing so the player no longer keeps the bottom toolbar gap in fullscreen landscape.
- Fixed the mobile overlay so hidden controls wake up on first tap instead of triggering actions underneath.
