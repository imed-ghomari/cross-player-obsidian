# Cross Player 1.2.1

This patch release improves audiobook support and cleans up player interactions on mobile.

Highlights:
- Added support for more browser-playable audiobook and audio formats, including `m4b`, `weba`, `aif`, `aiff`, and `caf`.
- Fixed the mobile progress strip so taps in that area cannot seek or trigger control actions while the overlay is hidden.
- Replaced the hard black bottom padding area with theme-aware spacing so the player blends correctly in light and dark themes.
