# Cross Player 1.1

This release focuses on everyday playback quality, especially on mobile.

Highlights:
- Added a safer `Clean Consumed Media` action in the queue with a confirmation dialog.
- Added optional wrapped queue text for long media names.
- Improved the mobile overlay with a coherent seek bar, visible playback times, and fullscreen access directly from the progress strip.
- Improved mobile fullscreen and tap-to-show overlay behavior.
- Added audio boost and sound normalization options for quiet media.
- Added persistent consumption statistics stored in compact daily buckets.
- Tuned the progress strip to follow the active Obsidian theme better in light and dark mode.
